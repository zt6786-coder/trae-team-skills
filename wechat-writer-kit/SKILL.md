# wechat-writer-kit - 公众号全流程创作技能

## 核心定位
内容创作者高频榜常客，覆盖公众号文章全生命周期：选题策划 → 大纲 → 正文（金句/数据/案例/排版建议） → 标题5选1 → 封面图提示词 → 发布检查清单。支持「深度长文/干货/故事/观点」四大模式，配合 humanizer-zh 实现「像人写的」自然中文。

## 四大创作模式

### 模式一：深度长文
- **适用**：行业深度、趋势研判、方法论沉淀、案例复盘
- **结构**：钩子开场(300字) → 核心论点(3-5个) → 每论点「论据+数据+案例+金句」 → 升华总结(行动建议)
- **篇幅**：3000-8000字，阅读时长8-15分钟
- **排版**：小标题+金句卡片+数据图表+关键词加粗+分割线节奏控制

### 模式二：干货教程
- **适用**：工具测评、操作指南、模板分享、避坑指南
- **结构**：痛点直击 → 核心方案(分步骤) → 实操细节(截图/代码/配置) → 常见问题FAQ → 资源包获取方式
- **篇幅**：2000-5000字，强调可执行性
- **排版**：步骤编号+代码块+高亮提示+资源链接卡片

### 模式三：故事叙事
- **适用**：创始人故事、用户见证、项目复盘、职场观察
- **结构**：悬念开场 → 人物/场景铺垫 → 冲突升级 → 转折时刻 → 结局与启示
- **篇幅**：1500-4000字，情绪曲线设计
- **排版**：场景化描写+对话还原+心理独白+金句提炼

### 模式四：观点评论
- **适用**：热点追评、行业观点、价值主张、争议性话题
- **结构**：事件背景(100字) → 核心观点(一句) → 多维论证(逻辑/数据/类比/反驳) → 预判与建议
- **篇幅**：1000-3000字，时效性强
- **排版**：观点金句前置+分论点清晰+反方预判+互动引导

## 标准化工作流

### Step 1：选题策划
**输入**：账号定位、目标受众、近期热点、历史爆款数据、内容日历
**输出**：5-10个备选题目 + 每题「切入角度/目标读者/预估阅读量/制作难度/参考素材」
**智能能力**：
- 热点关联度分析：结合微信指数/百度指数/微博热搜/知乎热榜
- 选题去重：对比历史文章库，避免重复/自相矛盾
- 爆款特征匹配：标题结构/开头钩子/结构模板/金句密度

### Step 2：大纲生成
**输入**：确定选题 + 模式选择 + 篇幅预算
**输出**：逐节大纲（含每节字数预算、核心论点、需引用数据/案例、金句预埋位）
**关键点**：
- 黄金前300字钩子设计：悬念/冲突/反常识/利益相关/故事开场
- 逻辑骨架：总-分-总 / 问题-方案-效果 / 现象-本质-应用
- 节奏控制：每800-1000字设置一个「呼吸点」（金句/图表/故事/互动）

### Step 3：正文深度展开
**输入**：确认大纲 + 参考资料(可选) + 企业素材库(可选)
**输出**：完整正文 Markdown（含排版标记、图表占位、引用标注）
**核心能力**：
- 金句生成器：每节至少1句可独立传播的金句（对仗/反转/隐喻/数据可视化）
- 数据编织：自动检索权威数据源（统计局/行业协会/上市公司财报/研报），生成引用标注
- 案例匹配：从素材库/公开案例库检索最贴切案例，自动改写适配语境
- 排版建议内嵌：`[H2]标题` `[BOLD]加粗` `[QUOTE]金句卡` `[CHART]图表占位` `[IMG]配图建议` `[DIVIDER]分割线`

### Step 4：标题 5 选 1
**输入**：正文全文 + 账号历史标题风格 + 目标分发渠道(订阅号/服务号/视频号伴文)
**输出**：5个差异化标题 + 每标题「预估CTR/适合场景/风险提示」
**标题公式库**：
- 数字具体化："3个底层逻辑/5年/10倍/90%的人不知道"
- 反直觉/冲突："为什么越努力越焦虑/大厂裁员却招不满人"
- 利益承诺："从0到1搭建/手把手教/避开90%坑"
- 情绪共鸣："那个把XX做成了的人/我们都在经历的"
- 悬念悬而未决："最后才发现/直到看完这篇/真相只有一个"

### Step 5：封面图提示词
**输入**：正文核心意象 + 标题关键词 + 账号视觉风格
**输出**：3组 Seedream/Seedance 提示词（含构图/配色/风格/文字留白区域）
**模板**：
- 极简大图风："Minimalist hero image, [核心隐喻], negative space for title, [品牌色系], 16:9"
- 故事场景风："Cinematic scene, [关键场景描述], emotional lighting, [色调], text-safe area top/bottom"
- 数据可视化风："Clean data visualization, [核心数据点], isometric/flat style, [品牌色], professional"

### Step 6：发布检查清单
**输出**：标准化检查项（逐项勾选）
- 内容合规：敏感词/违规表述/版权图片/医疗/金融/法律红线
- 排版规范：H1/H2层级/字号行距/图片对齐/代码块/链接可点击
- SEO优化：关键词布局/标题含关键词/摘要设置/标签标注
- 互动设置：文末引导/赞赏/名片/相关阅读/评论区置顶
- 定时发布：最佳发布时段预测/预约发布/同步视频号/同步社群

## 企业素材库集成
- **material_kb 结构**：
  ```
  material_kb/
  ├── gold_quotes/      # 金句库（按主题/风格/长度标签）
  ├── cases/            # 案例库（公司/项目/人物/结果/可引用素材）
  ├── data/             # 数据源（来源/时间/指标/可视化建议）
  ├── templates/        # 文章模板（四大模式各3-5套）
  ├── banned/           # 禁用词/敏感词/竞品词库
  └── style_guide/      # 账号风格指南（语气/禁语/排版/封面规范）
  ```

## 协作接口
- **上游**：接收 research-guide 产出的深度调研/行业报告/竞品分析；接收 data-analysis 产出的数据图表/趋势图
- **下游**：输出标准化 Markdown 供 humanizer-zh 去AI味；输出图文包供 byted-seedream-image-generate 生成配图
- **并行**：可与 founder-marketing-copywriter 协同（创始人视角深度文 vs 运营视角干货文）

- 元数据：创作模式、目标篇幅、核心关键词、预估阅读时长、生成时间、版本号
- 版本管理：大纲版/初稿版/定稿版/发布版，含变更记录
