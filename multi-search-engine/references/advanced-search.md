# 国内搜索引擎高级搜索指南

## 百度搜索高级语法

### 基础语法
| 语法 | 说明 | 示例 |
|------|------|------|
| `site:域名` | 在指定网站内搜索 | `site:zhihu.com 人工智能` |
| `filetype:类型` | 搜索指定文件类型（pdf/doc/xls/ppt/rtf） | `filetype:pdf 2024年政府工作报告` |
| `intitle:关键词` | 标题中包含关键词 | `intitle:Python 入门教程` |
| `inurl:关键词` | URL中包含关键词 | `inurl:news 华为` |
| `"精确短语"` | 完整匹配引号内内容 | `"全栈工程师"` |
| `-关键词` | 排除包含该词的结果 | `苹果 -手机 -水果 -电影` |
| `关键词1 OR 关键词2` | 包含任意一个 | `新冠 OR 新冠肺炎` |

### 百度特有语法
| 语法 | 说明 | 示例 |
|------|------|------|
| `时间范围` | 搜索框下方时间筛选 | 一天/一周/一月/一年 |
| `书名号《》` | 搜索影视、书籍作品 | `《三体》` |
| `减号-` | 排除无关词汇 | `小米 -手机` |

### 百度搜索技巧
1. **找官方资源**：加 `site:gov.cn` 或 `官网`
2. **找文档资料**：加 `filetype:pdf` 或 `filetype:doc`
3. **找问答内容**：加 `site:zhihu.com` 或 `百度知道`
4. **找最新资讯**：选择"一天内"时间筛选
5. **排除营销内容**：加 `-推广 -广告 -营销`

## 必应中国高级语法

### 基础语法（与百度通用）
支持 site:、filetype:、intitle:、inurl:、""、-、OR 等通用语法。

### 必应特有功能
| 功能 | 说明 |
|------|------|
| `site:cn.bing.com/news` | 资讯搜索 |
| `site:cn.bing.com/images` | 图片搜索 |
| `site:cn.bing.com/videos` | 视频搜索 |
| `&ensearch=0` | 强制中文结果 |
| `&ensearch=1` | 切换国际版 |

### 必应时间过滤URL参数
```
过去24小时: &filters=ex1:"ez1"
过去一周:   &filters=ex1:"ez2"
过去一月:   &filters=ex1:"ez3"
过去一年:   &filters=ex1:"ez4"
```

## 搜狗搜索高级语法

### 微信搜索专项
搜狗是唯一能搜索微信公众号内容的引擎：
- 微信搜索入口：`https://weixin.sogou.com/weixin?type=2&query={query}`
- type=2：公众号文章
- type=1：公众号账号

### 搜狗语法
| 语法 | 说明 |
|------|------|
| `site:weixin.sogou.com` | 微信内容 |
| `insite:zhihu.sogou.com` | 知乎内容 |
| `"短语"` | 精确匹配 |
| `-词` | 排除 |

### 搜狗搜索技巧
1. **搜微信文章**：直接用微信搜索入口，或在搜狗中选"微信"标签
2. **搜知乎内容**：搜狗的知乎搜索比百度更全
3. **搜学术论文**：搜狗学术搜索有知网/万方整合

## 360搜索语法

支持通用语法 site:、filetype:、intitle:、""、- 等。
特点：安全软件相关内容、恶意网站警告比较全面。

## 神马搜索

主要面向移动端，在移动端浏览器中使用效果更好。
支持基础语法，适合搜索手机应用、移动服务相关内容。

## 国内搜索引擎对比

| 需求场景 | 推荐引擎 | 原因 |
|----------|----------|------|
| 中文综合信息 | 百度 | 索引量最大 |
| 中英双语/技术 | 必应CN | 结果质量高，垃圾少 |
| 微信公众号 | 搜狗微信 | 独家内容 |
| 知乎/问答 | 搜狗 | 知乎收录好 |
| 官方文件/PDF | 必应/百度+filetype | 文档搜索 |
| 最新新闻 | 百度资讯/必应资讯 | 时效性好 |
| 学术论文 | 百度学术/搜狗学术 | 学术资源整合 |

## 国内搜索常见问题

### 1. 搜索结果太多广告怎么办？
- 使用 `site:` 限定权威网站
- 加 `-推广 -广告 -营销` 排除商业词
- 优先选择必应，广告较少

### 2. 找不到想要的内容？
- 尝试换关键词（同义词、简称/全称）
- 使用精确短语搜索 `""`
- 换搜索引擎交叉搜索
- 使用时间筛选缩小范围

### 3. 搜索结果过时？
- 使用时间过滤（一天/一周/一月）
- 在关键词中加入年份 `2024 2025`
- 选择"最近"排序
