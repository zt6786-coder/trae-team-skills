# founder-marketing-copywriter - 创始人视角营销文案技能

## 核心定位
专为创始人/高管/个人IP打造的高阶文案技能，覆盖品牌宣言、融资官宣、产品发布、朋友圈日更、演讲稿、对内动员、对外背书等场景。核心优势：语气人设一致性保障（跨渠道/跨时间/跨话题），A/B版本对比决策支持。定位精准，适合个人IP与品牌背书同步。

## 核心场景库

### 1. 品牌宣言 / Manifesto
- **触发时刻**：品牌重塑、系列融资后、战略转型、周年庆、上市敲钟
- **结构**：Why(信仰/使命) → What(承诺/边界) → How(方法论/差异) → Call to Action(邀请共创)
- **语气关键词**：坚定、克制、有温度、有锋芒、可被引用
- **产出**：长版(官网/白皮书/内部手册)、中版(公众号/领英/即刻)、短版(slogan/海报/视频旁白)

### 2. 融资官宣 / Funding Announcement
- **触发时刻**：Pre-A/A/B/C/Pre-IPO/战略融资完成
- **结构**：里程碑意义 → 投资人选择理由 → 资金用途透明化 → 业务进展佐证 → 未来承诺
- **语气关键词**：谦逊不卑微、自信不浮夸、数据支撑、长期主义
- **合规门禁**：证监会/交易所规则、保密条款、前瞻性声明免责
- **多渠道适配**：官方稿(正式)、创始人朋友圈(真实)、员工内部信(动员)、投资人背书稿(互推)

### 3. 产品发布 / Product Launch
- **触发时刻**：新品首发、大版本迭代、重大功能上线、生态合作发布
- **结构**：用户痛点(故事化) → 产品洞察(本质) → 核心能力(差异化) → 体验细节(惊喜) → 可获得性(价格/渠道/时间)
- **语气关键词**：像产品经理一样思考、像用户一样感受、像工程师一样严谨
- **A/B测试维度**：功能导向 vs 价值导向 / 理性说服 vs 感性共鸣 / 专业术语 vs 大白话

### 4. 创始人朋友圈 / Personal Social
- **触发时刻**：日常思考、关键节点、行业观察、团队时刻、读书/运动/生活碎片
- **结构**：一个场景/一个洞察/一个金句/一张图(可选)
- **语气关键词**：真实、有血肉、有思考深度、不说教、留白
- **运营节奏**：建议周3-5条，黄金发布时段(早8-9/午12-13/晚21-22)，互动回复策略

### 5. 演讲稿 / Keynote Speech
- **触发时刻**：行业大会、内部全员会、路演融资、大学分享、媒体访谈
- **结构**：钩子开场(30秒) → 核心主线(1条分3个支撑点) → 故事/数据/金句交织 → 高潮升华 → 行动号召
- **语气关键词**：节奏感强、口语化、可视化语言、留有互动余地
- **配套产出**：讲稿全文 + 提词器版(分段/重音/停顿/眼神提示) + Q&A预判库

### 6. 对内动员 / Internal Rally
- **触发时刻**：年度启动会、季度复盘、危机时刻、组织变革、关键项目冲刺
- **结构**：现实直面(不回避困难) → 信念锚定(为什么能赢) → 战略聚焦(做什么/不做什么) → 个人连接(每个人的角色) → 仪式感收尾
- **语气关键词**：坦诚、共情、有方向感、可执行、让人想跟随

## 人设一致性引擎

### 人设画像构建
**输入**：创始人访谈(2-3小时) + 历史公开内容(演讲/文章/采访/朋友圈) + 团队360反馈
**输出**：人设画像卡
- **核心价值观**：3-5个不可妥协的信念
- **语言指纹**：高频词/句式偏好/修辞习惯/禁用词/标点风格
- **情绪基调**：理性/感性/幽默/严肃/温和/锐利 的配比
- **视角高度**：战略/战术/执行/哲学 的切换规则
- **叙事原型**：探索者/统治者/照顾者/英雄/智者/普通人

### 一致性守门机制
- **生成时**：每段文案自动打分（人设符合度/语气连贯性/术语准确性）
- **审核时**：提供「偏离预警」高亮（用词不像/语气突变/逻辑跳跃/价值观冲突）
- **迭代时**：人设画像随新内容持续微调，版本化管理

## A/B 版本对比框架

| 维度 | 版本A | 版本B | 决策建议 |
|------|-------|-------|----------|
| 核心切入点 | 痛点直击 | 愿景引领 | 视受众成熟度选 |
| 语气风格 | 理性专业 | 温暖真诚 | 视渠道属性选 |
| 证据类型 | 数据/案例 | 故事/隐喻 | 视传播目标选 |
| 行动号召 | 立即体验 | 关注长期 | 视转化路径选 |
| 预估效果 | 点击率高 | 完读率高 | 视KPI权重选 |

**决策支持**：
- 多维雷达图可视化对比
- 历史同类文案效果回溯
- 专家/团队盲测投票接口

## 协作接口
- **上游**：接收 research-guide 产出的行业趋势/竞品定位/用户洞察；接收 data-analysis 产出的业务数据/增长曲线
- **下游**：输出标准化文案包供 humanizer-zh 精细润色；输出演讲稿供 byted-seedance-video-generate 生成数字人视频/练习视频
- **并行**：可与 wechat-writer-kit 协同（创始人深度文 vs 运营干货文）；可与 quick4prd 协同（商业计划书/融资BP文案段）
